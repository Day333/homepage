### ğŸš€ Quick Start

1.  **Clone/Copy**: Copy the `AcaNova-X` folder to your web server or GitHub Pages repository.
2.  **Personalize**:
    *   Edit `index.html`: Update your name, bio, affiliation, and contact links.
